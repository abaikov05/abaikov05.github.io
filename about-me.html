FAQ

Yes, I usually only write about things I've read, watched, or studied. Sometimes I test things just for practice, like now.

Because it's made for homework and testing out different features. Maybe I'll make it prettier in the future - if I have time.
Can I use your materials?
Sure. If anyone finds my code or text useful, feel free to use it. Please cite the source if you copy it in its entirety.
Will you continue to develop this blog?
For now, this is just a test project. But if it becomes interesting or useful, I may continue.
